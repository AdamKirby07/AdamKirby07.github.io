How to use BISCUIT MCU™ and BISCUIT MCU MINI™ with MPLAB IDE - Pre-release Tutorial

This tutorial will teach you how to use BISCUIT MCU™ and BISCUIT MCU MINI™ with MPLAB IDE, a great resource for anyone learning about microcontroller programming.

Prerequisites:

- MPLAB (from Microchip)

- XC8 Compiler (from Microchip)


Please ensure that these have been installed before proceeding with the next steps. :)

Code Offset:

A code offset of 0x1000 should be applied.


This can be done by:

  • Right-click the project name and select properties.
  • Select the XC8 linker under XC8 global options.
  • In the options category, select additional options and enter the code offset of 0x1000.
  • Click apply, then OK.

Pin Assignments:

I recommend assigning the BISCUIT MCU™ name of each pin in MPLAB to make things easier.


MPLAB C code:
#define 1 RA5
#define 2 RA4
#define 3 RC5
#define 4 RC4
#define 5 RC3
#define 6 RC2
#define 7 RC1
#define 8 RC0
